# Noortepoliitika Automatiseerimise Keskus

See repo sisaldab tööriistu ja juhiseid, mis aitavad automatiseerida:
- Meediaseiret ja mainekujunduse analüüsi.
- Sidusvaldkondade sisendite töötlemist harta koostamisel.
- Rutiinset bürokraatiat.

## Struktuur
- `/scripts` - Google Apps Scripti koodid.
- `/prompts` - AI süsteemijuhised.
- `/docs` - Täiendav dokumentatsioon.
